# Random Artwork Generator

A lightweight single-page webpage that generates random artwork directly in your browser using the HTML Canvas API.

## Included art modes

- Watercolor-style painting
- Canvas texture painting
- Still life scene
- Futuristic / modern city scene (skyscrapers, lights, vehicle-like elements)

## Run locally

```bash
python3 -m http.server 4173
```

Then open:

- `http://127.0.0.1:4173`
